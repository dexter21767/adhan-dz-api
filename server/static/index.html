Welcome to Adhan DZ API

This API provides prayer times for cities in Algeria. Below are the available endpoints and their usage.

Available Endpoints

GET /cities

Retrieve a list of cities with their IDs and names.

Response Example:

[
  {
    "_id": 1,
    "name": "Adrar",
    "parentId": null
  },
  {
    "_id": 2,
    "name": "Chlef",
    "parentId": null
  },
  ...
]
Try it out

GET /prayerTimes

Retrieve prayer times for a specific city and date range.

Query Parameters:

Parameter Type Required Description
cityId integer No Use ParentId if present, or _id if ParentId = null
startDate string No The start date in YYYY-MM-DD format (defaults to today if not provided)
endDate string No The end date in YYYY-MM-DD format (if not provided, only returns data for the start date)

Response Example:

[
  {
    "date": "2025-03-15",
    "fajr": "05:14",
    "sunrise": "06:40",
    "dhuhr": "12:55",
    "asr": "16:23",
    "maghrib": "19:10",
    "isha": "20:36"
  },
  {
    "date": "2025-03-16",
    "fajr": "05:12",
    "sunrise": "06:39",
    "dhuhr": "12:54",
    "asr": "16:24",
    "maghrib": "19:11",
    "isha": "20:37"
  }
]

Note: If the end date is more than 30 days after the start date, it will be changed to 30 days after the start date.

Try it out

Error Handling

All errors will be returned with appropriate HTTP status codes and a JSON body containing an error message.

{
  "error": "Invalid date format. Use YYYY-MM-DD."
}

Rate Limiting

This API is rate-limited to 100 requests per hour per IP address. If you exceed this limit, you'll receive a 429 status code.
